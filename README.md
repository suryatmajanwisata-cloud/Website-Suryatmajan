# Website-Suryatmajan

Website Kampung Suryatmajan
Website ini merupakan halaman informasi sederhana mengenai Kampung Suryatmajan. Dibuat menggunakan HTML, CSS, dan Bootstrap untuk tampilan yang responsif dan mudah diakses. Project ini adalah website statis yang bertujuan untuk memperkenalkan Kampung Suryatmajan secara online. Di dalam website ini menampilkan konten dasar seperti gambaran umum tentang kampung, lalu fasilitas apa saja yang ada disana, dan menampilkan UMKM apa saja yang ada disana.
